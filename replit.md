# 출근일보 변환기 (Attendance Converter)

## 프로젝트 개요
엑셀 출근일보를 상세명세서로 자동 변환하는 풀스택 웹 애플리케이션입니다.

## 핵심 기능
- **파일 업로드**: 드래그앤드롭 또는 클릭으로 엑셀 파일(.xlsx, .xls) 업로드
- **시트 선택**: 업로드된 파일에서 변환할 시트 선택
- **자동 변환**: 성별/근무시간에 따른 공수, 단가, 계 자동 계산
- **다운로드**: 변환된 상세명세서 엑셀 파일 자동 다운로드

## 기술 스택
- **Frontend**: React 18 + Vite + Tailwind CSS + shadcn/ui
- **Backend**: Node.js + Express.js
- **Excel 처리**: ExcelJS
- **파일 업로드**: Multer (memoryStorage)

## API 엔드포인트
- `GET /api/health` - 서버 상태 확인
- `POST /api/sheets` - 업로드된 파일의 시트 목록 반환
- `POST /api/convert` - 파일 변환 및 다운로드

## 출력 파일 형식
- **파일명**: YY-MM-회사명-상세명세서.xlsx
- **탭명**: MM월 상세명세서
- **컬럼**: NO., 일정, 성별, 성명, 기본, 연장, 공수, 단가, 연장, 계, 조율OT

## 로컬 실행
```bash
npm run dev
```
